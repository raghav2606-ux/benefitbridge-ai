# 🚀 BenefitBridge AI

BenefitBridge AI is an AI-powered platform that helps citizens discover government schemes they are eligible for through personalized recommendations and simplified eligibility analysis.

## ✨ Features

- 🤖 AI-powered scheme recommendations
- 📋 Personalized eligibility checking
- 📄 Document guidance
- 🌐 Official government application links
- 📱 Responsive and modern UI

## 🛠 Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

### Backend
- FastAPI
- Python
- Pydantic

## 🚀 Run Locally

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 9000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🎯 Purpose

BenefitBridge AI simplifies access to government welfare schemes by providing personalized recommendations, clear eligibility insights, and direct application guidance.
